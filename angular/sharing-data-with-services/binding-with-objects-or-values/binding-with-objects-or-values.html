Binding to service objects or static values (numeric, string etc.)

This scenario observes how Angular's ng-bind function can watch values from a service, comparing what happens when we bind a local scope variable to a service's *object*, and a service's *primitive value*.

The result displays that ng-bind is capable of watching changes to a service's object, but *NOT* a primitive value.
