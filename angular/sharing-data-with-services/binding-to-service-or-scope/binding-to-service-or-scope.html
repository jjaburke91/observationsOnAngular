Binding To Direct To A Service Function

View the console log to see how frequently the service function is being executed.

The function gets executed *every time* the model changes. This indicates the the digest cycle is watching the result of that function, executing it on every cycle - inefficient!

Instead, we can bind the service object to a scope variable

{{sharedService.getModelObjectFunction()}} - {{sharedService.getModelObjectFunction()}} - Directly executes service function to retrieve service object.
